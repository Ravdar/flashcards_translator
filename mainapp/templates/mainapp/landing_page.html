Frequently asked questions

What is InstantCard?


InstantCard is flashcard application. What distinguishes it from other such applications is the function of automatic card creation when translating text.



What features does it have?


It can create flashcards while translating, translate text without creating flashcards, add, edit and delete decks and flashcards. Then you can also study your flashcards using an advanced algorithm designed to optimize learning. For now, basic statistics are also available for viewing, an option that will be much more extensive in the future.



Is there a mobile app?


A mobile application is planned, but it is not yet available. Instead, you can use the browser version on your phone at the moment, because the AnkiFlashcards website is fully responsive!



Are there any new features planned?


Yes, there are many upcoming features, including: much more advanced learning statistics, the ability to create non-language decks and cards, the ability to add images to cards, importing decks from other applications such as Anki, different types of flashcards (such as gap), browser extension and much more!



What kind of algorithm do you use for material review?


AnkiFlashcards uses improved SM-2 algorithm. It is based on research into long-term memory, and is a practical application of the spaced repetition learning method that has been proposed for efficient learning.
